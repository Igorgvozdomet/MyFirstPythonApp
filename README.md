# Приложение «Погода» на Flet
Кроссплатформенное приложение для просмотра погоды, написанное на Python с использованием фреймворка Flet. Работает на Windows, macOS, Linux, Android, iOS и в вебе.

Пользователь вводит название города, нажимает кнопку — приложение отправляет запрос к OpenWeatherMap API и отображает текущую температуру в градусах Цельсия. Интерфейс адаптивный: поле ввода, кнопка и текстовый блок с результатом расположены по центру. Реализовано переключение между светлой и тёмной темами (иконка солнца в заголовке).

Технологии: Python, Flet (кроссплатформенный GUI), Requests (HTTP-запросы), OpenWeatherMap API. В проекте отработаны навыки работы с API, обработки пользовательского ввода, управления состоянием интерфейса и изменения темы оформления.

Этот проект стал моим первым опытом в разработке приложений с графическим интерфейсом и показал, как быстро можно создавать удобные кросс-платформенные решения на Python.
